นโยบายความเป็นส่วนตัว — SE-Billing Extension

อัพเดทล่าสุด: 6 พฤษภาคม 2026 (Last updated: 2026-05-06)

1. ข้อมูลที่ extension เก็บ

SE-Billing Chrome Extension ทำงานบนหน้าเว็บ https://cloud.isurvey.mobi/* เท่านั้น และเก็บข้อมูลต่อไปนี้:

  • ข้อมูลฟอร์มที่ user กรอก: จังหวัด/อำเภอ/ตำบลของงานสำรวจ, รหัสประเภทเคลม, ชื่อพนักงานสำรวจ, ค่าบริการ และ field ที่เกี่ยวข้องในแบบฟอร์ม
  • ข้อมูลผู้ใช้งาน: ชื่อ inspector ที่ login (อ่านจาก header ของหน้า I Survey)
  • การตั้งค่า extension: Server URL, API token, รายการจังหวัดที่ user เลือกใช้งาน — เก็บใน chrome.storage.local ของเครื่องผู้ใช้

Extension ไม่เก็บ: รหัสผ่าน, ข้อมูลทางการเงินส่วนตัว, ข้อมูลในแท็บอื่นที่ไม่ใช่ I Survey, ประวัติการเข้าชมเว็บไซต์

2. วัตถุประสงค์

ข้อมูลที่บันทึกใช้สำหรับ:

  • คำนวณและเติมค่าบริการลงในฟอร์มอัตโนมัติตามตารางเรทของบริษัท
  • เก็บ audit log ของงานที่ user กรอกเพื่อทำบัญชีค่าบริการรายเดือน
  • วิเคราะห์ภาพรวมการใช้งานเพื่อปรับปรุงอัตราค่าบริการ

3. การส่งข้อมูลไปยัง server

เมื่อผู้ใช้กดปุ่ม "ยืนยันการตรวจสอบ" บนหน้า I Survey, extension จะส่ง snapshot ของฟอร์ม (ฟิลด์ตามข้อ 1) ไปยัง backend server ที่กำหนดใน Settings (default: https://billing.sesurvey.cloud) ผ่าน HTTPS โดยมี Bearer token ป้องกันการเข้าถึง

การส่งข้อมูลเกิดเฉพาะตอนกดปุ่มยืนยันเท่านั้น ไม่มีการส่งข้อมูลแบบ background polling หรือ telemetry อื่นๆ

4. การเก็บรักษา

  • ข้อมูลใน chrome.storage.local เก็บอยู่บนเครื่อง user เท่านั้น — ลบได้โดย uninstall extension
  • ข้อมูลที่ส่งไป server เก็บใน SQLite ของ SE-Billing โดยมีระยะเวลาเก็บไม่จำกัด (สำหรับ audit log)
  • หากต้องการให้ลบข้อมูล server-side ติดต่อ admin ผ่าน sesurvey.ai@gmail.com

5. การแชร์กับบุคคลที่สาม

SE-Billing ไม่ขายหรือแชร์ข้อมูลของ user ให้บุคคลที่สามใดๆ ทั้งสิ้น ข้อมูลใช้ภายในบริษัทเพื่อวัตถุประสงค์ตามข้อ 2 เท่านั้น

6. ความปลอดภัย

  • การส่งข้อมูลทั้งหมดผ่าน HTTPS (TLS 1.2+)
  • API ป้องกันด้วย Bearer token
  • Server เก็บใน Hostinger VPS ที่ user ควบคุมเอง ไม่ผ่าน third-party SaaS

7. สิทธิ์ของผู้ใช้

ผู้ใช้สามารถ:

  • ขอดูข้อมูลที่บันทึกของตน
  • ขอลบข้อมูล (right to erasure)
  • Uninstall extension ได้ทุกเมื่อ — chrome.storage.local จะถูกลบอัตโนมัติ

ติดต่อขอใช้สิทธิ์ที่ sesurvey.ai@gmail.com

8. การเปลี่ยนแปลงนโยบาย

หากมีการเปลี่ยนแปลงนโยบายความเป็นส่วนตัว จะแจ้งผ่าน update ของ extension และอัพเดทวันที่ในหน้านี้
