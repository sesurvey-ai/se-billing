I Survey Auto-Fill Helper — Settings

การตั้งค่าการเชื่อมต่อกับ backend server (ที่จัดการเรตราคา + เก็บ log การกรอกฟอร์ม)

เช่น http://localhost:3200 (เครื่องเดียวกัน) หรือ http://192.168.1.50:3200 (เครื่องอื่นใน LAN เดียวกัน) — ห้ามมี / ปิดท้าย

เปิดหน้า server (เมื่อตั้งค่า URL ถูกแล้ว):

  • หน้า เรทราคา (อ่านอย่างเดียว)
  • หน้า Admin (เพิ่ม/แก้/ลบ)
  • หน้า รายละเอียด (ข้อมูลที่จับจากฟอร์ม)
